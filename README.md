## 项目特性

- **智能匹配算法**：根据技能和需求精准推荐
- **招募信息发布**：快速创建和展示招募信息
- **个人简历管理**：展示个人优势与经历
- **实时反馈系统**：及时获取组队动态

## 技术栈

- **框架**: Vue 3
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **语言**: JavaScript

## 环境要求

- Node.js ≥ 16.x
- npm 或 yarn

## 安装与运行

1. **进入项目目录**
   ```bash
   cd 智能组队小助手
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```
